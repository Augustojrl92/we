¿Qué es CGI?

CGI (Common Gateway Interface) permite que el servidor web ejecute programas externos y devuelva el resultado al navegador. Es perfecto para crear páginas dinámicas.

Scripts disponibles:

  • simple.py - El script más básico posible
  • hello.py - Muestra información básica del servidor
  • hello.php - Versión PHP simple
  • form_handler.py - Procesa formularios
  • test.py - Script completo con todas las variables
